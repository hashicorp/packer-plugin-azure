# Azure Data Sources

The Azure plugin is able to fetch data from Azure. To achieve this, the plugin comes with data sources to retrieve keyvault secrets.
Packer supports the following data sources at the moment:

- [azure-keyvault-secret](/docs/datasources/azure/keyvault-secret) - Retieve secret value and additional information from Azure Keyvault

## How to use this plugin

From Packer v1.7.0, copy and paste this code into your Packer configuration to install this plugin.
Then, run [`packer init`](/docs/commands/init).

```hcl
packer {
  required_plugins {
    azure = {
      version = ">= 0.0.2"
      source  = "github.com/hashicorp/azure"
    }
  }
}
```
